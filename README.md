🎨 Color Picker Web App

📌 Overview

This is a simple Color Picker web application developed using HTML, CSS, and JavaScript. It allows users to change the background color of the page by clicking on different color buttons.

🚀 Features

* 🎯 Change background color with a single click
* 🎨 Multiple color options available
* 🔄 Dynamic update of selected color code
* 💡 Beginner-friendly project to understand DOM manipulation


🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript


 📂 Project Structure
├── index.html
├── colourpicker.css
├── colourpicker.js



## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Click on the color buttons to see the changes


📸 Output

The background color changes dynamically when a user clicks on any color button, and the selected color code is displayed on the screen.



🎯 Learning Outcomes

* Understanding DOM manipulation
* Handling button click events
* Updating UI dynamically using JavaScript

📌 Future Improvements

* Add more color options
